## Ollama LLM Projects

Collection of local LLM projects built using Ollama.

### Setup

```bash
git clone <repo>
cd <repo>
ollama pull <model-name>
```

### Projects

* `Bengali Sentiment Analyzer` using Sailor2
* `Image Caption Generator` using Moondream
* `Text Summarizer` using Llama3.2:1b

### Tech Stack

Ollama, Python, FastAPI, Streamlit
